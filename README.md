# FP.048 - (P) Administración de servidores empresariales

Repositorio de documentación del proyecto grupal (132 horas). El objetivo es crear una infraestructura de dominio para centralizar la gestión de recursos y servicios, integrar sistemas operativos libres y propietarios, automatizar tareas mediante scripting y desplegar un sistema gestor de bases de datos con estrategias de mantenimiento y copias de seguridad.

## Alcance del proyecto
- Producto 1: Administración remota del servicio de directorio (diseño, AD DS, DNS, DHCP, acceso remoto).
- Producto 2: Servicios y recursos de red (OUs, usuarios, grupos, permisos, integración Windows/Linux, carpetas e impresión).
- Producto 3: Automatización de tareas y procesos (PowerShell: gestión de usuarios, monitorización, ficheros, procesos y tareas programadas).
- Producto 4: Implantación de base de datos (SGBD, autenticación con AD, backup/restore y acceso web de demostración).
